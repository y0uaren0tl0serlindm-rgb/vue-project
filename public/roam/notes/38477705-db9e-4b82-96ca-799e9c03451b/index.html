  • 以共享打印机的方式进行连接
    • 主机有线连接打印机
    • 网络切换到专用网络
    • 控制面板-网络与共享-高级共享设置-所有网络-取消密码保护的共享
    • (可选)确认一下服务正在运行 Win+R 输入 services.msc Server Workstation Function Discovery Provider Host Function Discovery Resource Publication SSDP Discovery Upnp Device Host Print Spooler
    • (可选)检查防火墙是否放行 Win+R 输入 control firewall.cpl 允许应用或功能通过 Windows Defender 防火墙 勾选 文件和打印机共享 和 网络发现
    • 0x0000011b 错误码，处理方法： Win + R 输入 regedit 找到路径 HKEYLOCALMACHINE\System\CurrentControlSet\Control\Print 右键空白处，新建 DWORD(32 位)值 名称填写 RpcAuthnLevelPrivacyEnabled 数值设定 0 重启打印服务 net stop spooler net start spooler
