1. 训练过程优化:欠拟合、过拟合、测试与正则化

1.1. 模型

欠拟合：使用对于数据集来说过于简单的模型 合理拟合：使用对于数据集来说复杂度合适的模型 过拟合：使用对我们数据集来说过于复杂的模型

1.2. 训练集中

欠拟合模型表现不佳（训练误差很大） 好的模型表现会很好（训练误差很小） 过拟合模型表现非常好（训练误差非常小）

1.3. 测试集中

欠拟合模型，不佳（误差，很大） 好的模型，很好（误差，很小） 过拟合模型，不佳（误差，很大）

绝对不能用测试数据进行训练

3. 一种数值方法来决定模型的复杂度：模型复杂图

这个图纵坐标是误差，横坐标是复杂度，描述两个误差 训练误差、验证误差
如果只有训练误差，我们难免陷入到追求越来越小的误差 于是我们有了验证误差，这可以让我们注意到，在模型复杂度不断上升的过程中，虽然训练误差不断减小但是验证误差不断增大 我们可以判断这就是过拟合的趋势 所以我们通过两个误差综合判断保留的模型的复杂度 而非一直追求小训练误差

5. 衡量模型复杂度:L1 和 L2 规范

模型1: \(\hat{y} = 2x_3 + 1.4x_7 - 0.5x_7 + 8\)
模型2: \(\hat{y} = 22x_1 - 103x_2 - 14x_3 + 109x_4 - 93x_5 + 203x_6 + 87x_7 -55x_8 + 378x_9 - 25x_10 + 8\)
指标，判断模型2比模型1复杂得多

5.1. 系数的绝对值之和

\(|x| + |1.4| + |0.5| = 3.9\)
\(|22| + |-103| + |-14| + |109| + |-93| + |203| + |87| + |-55| + |378| + |-25| = 1{,}089\)

5.2. 系数平方和

\(2^2 + 1.4^2 + (-0.5)^2 = 6.21\)
\(22^2 + (-103)^2 + (-14)^2 + 109^2 + (-93)^2 + 203^2 + 87^2 + (-55)^2 + 378^2 + (-25)^2 = 227{,}131\)
