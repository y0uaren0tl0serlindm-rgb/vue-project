2. 安装流程

2.1. 检查硬件支持

#检查硬件支持,验证内核启用了虚拟化
egrep -c '(vmx|svm)' /proc/cpuinfo
#确认 KVM 内核模块能加载
lsmod | grep kvm

本机返回 

sisyphus@sisyphus:~$ egrep -c '(vmx|svm)' /proc/cpuinfo
24
sisyphus@sisyphus:~$ lsmod | grep kvm
kvm_intel             487424  0
kvm                  1409024  1 kvm_intel
irqbypass              12288  1 kvm

2.2. KVM生态

你的命令行/工具 ← virsh、virt-install、virt-manager ↓ 管理守护进程 ← libvirtd (统一管理接口) ↓ 虚拟机模拟器 ← QEMU (实际跑虚拟机的程序) ↓ 内核虚拟化加速 ← KVM (内核模块，已加载✓) ↓ CPU 硬件虚拟化 ← VT-x (你的 i5-12490F✓)

安装以上各个包 qemu-kvm libvirt-daemon-system libvirt-clients bridge-utils virtinst 

sudo apt update
sudo apt install qemu-kvm libvirt-daemon-system libvirt-clients bridge-utils virtinst

2.3. Download Ubuntu Server ISO

虚拟机 分配三样资源: 1.计算资源:CPU核数、内存–从宿主机分一部分给它 2.存储:一个磁盘镜像文件，从虚拟机角度看就是他的硬盘 3.安装介质:Ubuntu的ISO镜像，相当于光盘，第一次启动从这里装系统

资源 分配 说明
==== ==== =================================
CPU 4核 本机有12核，留出余量
内存 4GB 编内核够用，本机留下27GB
磁盘 40GB 装系统5G，源码+编译产物约25GB，留出余量

下载Ubuntu Server ISO 

sudo mkdir -p /var/lib/libvirt/iso
cd /var/lib/libvirt/iso
sudo wget https://releases.ubuntu.com/24.04/ubuntu-24.04.3-live-server-amd64.iso

2.4. 安装虚拟机

#创建虚拟机
virt-install
#退出虚拟机： Ctrl + ]
